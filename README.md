Bài tập cho anh em để t xem anh em hiểu git đến đâu nhé <p>
B1 : Checkout về commit có tên của anh em mà t đã commit <P>
B2 : Tạo branch riêng với tên anh em (ví dụ như t là Khanh_Dev) <p>
B3 : Tạo Folder riêng với tên anh em (ví dụ như t là Khanh_Folder) <p>
B4 : Tạo file riêng với tên anh em (ví dụ như t là Khanh.txt) <p>
B5 : Viết nội dung file bất kỳ <p>
B6 : Commit với nội dung đã hoàn thành bài (ví dụ như t là Khanh AC Test) <p>
